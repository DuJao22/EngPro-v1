CivilSaaS

Sistema Completo para Gestão de Projetos de Engenharia Civil

Sobre o Sistema

O CivilSaaS é uma plataforma completa desenvolvida especificamente para engenheiros civis, oferecendo todas as ferramentas necessárias para o gerenciamento eficiente de projetos de construção.

Principais Funcionalidades:
  • Gestão de Projetos
  • Controle de Orçamento
  • Licenças e Alvarás
  • Segurança e Incidentes
  • Gestão de Fornecedores
  • Treinamentos e Capacitação
  • Sustentabilidade e Emissões
  • Gestão de Riscos
  • Compliance e Documentos
  • Medições de Campo IoT
  • Relatórios Detalhados
  • Notas Rápidas
Tecnologias Utilizadas
Backend:
  • Python Flask
  • SQLite3
  • Werkzeug Security
  • File Upload Support
Frontend:
  • Bootstrap 5
  • Font Awesome
  • Chart.js
  • Vanilla JavaScript
